## Repository for the Cold Spring Harbor Laboratory, Imaging of Nervous Systems Course.


Maintained by Adam S. Charles (adamsc@jh.edu) and Esther Whang (esther.m.whang@gmail.com)


Includes setup, demo notebooks, and other helpful resources for learning.

#### SETUP

Please ensure that you have conda set up on your device (miniconda installation instructions can be found in the course manual). The following environments were used during the 2025 course:

* caiman.yml 
* CSHL2025.yml

The caiman.yml is essentially the same as the one you would get from the caiman installation instructions, but there were one or two extra packages that was installed during the course that was not properly accounted for. Installation may be easier if you follow the caiman installation instructions. I suggest following their provided guide (https://github.com/flatironinstitute/CaImAn?tab=readme-ov-file#quick-start-route-a), then figuring out which packages are missing by running final_project_caimain.ipynb. 


CSHL2025.yml contains all the packages to run the non-CaImAn related notebooks.
